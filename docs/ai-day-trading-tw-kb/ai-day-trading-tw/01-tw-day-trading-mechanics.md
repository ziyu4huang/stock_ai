# 01 — 台股當沖機制 (TW Day Trading Mechanics)

## 1. 什麼是當沖 (Day Trading / 當日沖銷)?

當沖 = 在同一交易日內買進並賣出（或賣出並買進）同一檔股票。關鍵優勢：**不留過夜部位 (No Overnight Position)**，不需要支付全额股價。

### 兩種模式：
1. **先買後賣 (Buy First, Sell Later)：** 早盤買進，收盤前賣出。股價上漲時獲利。
2. **先賣後買 (Sell First, Buy Later)：** 早盤先賣出，收盤前買回。股價下跌時獲利。需股票價格高於前日收盤價 (平盤上)。

### 核心機制：
- 你**不需要支付全额股價**。券商 (Broker) 在收盤時自動沖銷 (Netting) 你的買賣部位
- 你只需要支付：**手續費 (Commission) + 交易稅 (Tax) + 買賣價差 (P&L)**
- 買進價 > 賣出價：你支付差額 (虧損)
- 賣出價 > 買進價：你收到差額 (獲利)

## 2. 可當沖標的 (Eligible Securities)

並非所有股票都能當沖。台交所 (TWSE) 每日公布可沖銷標的清單：
- **查詢網址：** https://www.twse.com.tw/zh/trading/day-trading.html
- 多數大型股 (Large-cap) 可當沖：台積電 2330、鴻海 2317、聯發科 2454 等
- ETF（0050、0056 等）可當沖
- 櫃買中心 (TPEx / OTC) 也有當沖標的清單
- 創新板 (TIB) 自 2025/11/17 開放當沖

### 限制條件：
- **平盤下不得融券賣出：** 股價低於前日收盤價時，不能「先賣後買」
- 部分股票可能被暫停「先賣後買」資格
- 台交所公布「暫停先賣後買當日沖銷交易標的預告表」

## 3. 費用結構 (Fee Structure) 詳解

### 3.1 券商手續費 (Broker Commission)
- **法定上限：** 成交金額的 0.1425%
- **電子交易折扣 (e-Trading Discount)：** 通常 6-8 折
- **議價空間：** 交易量大可議價至 0.02-0.06%
- **最低手續費：** 每筆 NT$20（部分券商可免除）
- 買進和賣出**各收一次**

### 3.2 證券交易稅 (Securities Transaction Tax)
- **一般稅率：** 0.3%（僅賣出時收取）
- **當沖優惠稅率：** **0.15%**（減半）— 僅限當日完成買賣
- 當沖降稅自 2017/4/28 實施，已多次延長
- **目前期限：** 延長至 **2027/12/31**
- 若當日未完成沖銷，則按 0.3% 課稅

### 3.3 成本計算範例

**交易：** 買進 1張 (1000股) 台積電 2330 @ NT$1,810，賣出 @ NT$1,820（+1 檔 NT$10）
> 註：NT$1,810 為 2026/4/2 收盤價。1 檔 = NT$5（NT$1,000 以上檔位），但此例假設價格移動 2 檔 = NT$10。

```
成交金額：NT$1,810,000 (買) + NT$1,820,000 (賣)
買進手續費：  1,810,000 × 0.1425% = NT$2,579
賣出手續費：  1,820,000 × 0.1425% = NT$2,594
當沖交易稅：  1,820,000 × 0.15%  = NT$2,730
───
總費用：      NT$7,903
毛利 (Gross Profit)：    NT$10,000
淨利 (Net Profit)：      NT$2,097
```

**議價費率 (0.03%) 範例：**
```
買進手續費：  1,810,000 × 0.03% = NT$543
賣出手續費：  1,820,000 × 0.03% = NT$546
當沖交易稅：  1,820,000 × 0.15% = NT$2,730
───
總費用：      NT$3,819
淨利 (Net Profit)：      NT$6,181
```

### 3.4 損益兩平價差 (Breakeven Spread)

以議價費率 0.03% 計算，來回總成本 (Round-trip Cost)：
- 來回總成本 ≈ 買手續費 + 賣手續費 + 當沖交易稅 = 0.03% + 0.03% + 0.15% = **0.21%**

以不同股價計算損益兩平所需的移動：
| 股價 | 檔位 (Tick Size) | 0.21% 需移動 | 幾檔 (Ticks) | 損益兩平難度 |
|------|-----------------|-------------|-------------|-------------|
| NT$50 | NT$0.05 | NT$0.11 | ~2 檔 | 低 |
| NT$200 | NT$0.50 | NT$0.42 | ~1 檔 | 低 |
| NT$500 | NT$1.00 | NT$1.05 | ~1 檔 | 中 |
| NT$1,000 | NT$5.00 | NT$2.10 | ~1 檔 | 中 |
| NT$1,810 (2330) | NT$5.00 | NT$3.80 | ~1 檔 | 中 |
| NT$5,000 | NT$5.00 | NT$10.50 | ~2 檔 | 偏高 |

> **關鍵觀察：** NT$1,000 以上股票的檔位一律 NT$5，但價格越高、檔位的百分比越小（如 2330 的 1 檔 = 0.28%），反而越容易損益兩平。真正難的是 NT$5-10 的低價股，1 檔 = 1-10%。

## 4. 交割機制 (Settlement Mechanics)

- **T+0 當日沖銷：** 部位當日沖銷 (Netting)，不需實際交割
- **T+2 交割：** 現金交割 (Cash Settlement) 在成交日後第 2 個營業日完成
- **若收盤前未沖銷：** 強制 T+2 交割，交易稅恢復 0.3%
- **違約交割風險 (Default Risk)：** 若無法支付，券商有權強制賣出，且面臨法律後果

## 5. 當沖保證金與額度 (Margin Requirements)

### 5.1 現股當沖
- **先買後賣 (Buy First)：** 不需保證金 (Margin)
- **先賣後買 (Sell First)：** 需開立信用交易帳戶 (融資融券帳戶)
- **開戶條件：** 需 NT$30萬+ 存款，通過財力審查
- **建議最低資金：** NT$30-50萬以上才有意義

### 5.2 沖不掉怎麼辦 (Failed Close)?
- 券商會代為借券 (Borrow Shares) 完成交割
- 需支付借券費 (Borrowing Fee)：年利率約 0.1-1%
- 最壞情況：隔日以市價強制買回
- **風險管理：** 務必設定停損 (Stop-Loss)，13:00 前強制平倉

## 6. 台股微結構 (Market Microstructure)

### 6.1 交易階段 (Trading Phases)
| 時間 | 階段 | 機制 |
|------|------|------|
| 08:30-09:00 | 開盤前撮合 (Pre-market Auction) | 收集委託，集合競價決定開盤價 |
| 09:00-13:25 | 盤中逐筆交易 (Continuous Trading) | 價格時間優先 (Price-Time Priority) |
| 13:25-13:30 | 收盤集合競價 (Closing Auction) | 集合競價決定收盤價 |
| 14:00-14:30 | 盤後定價 (After-hours Fixed Price) | 以收盤價交易，不適用當沖 |

### 6.2 委託單類型 (Order Types)
- **ROD (Rest of Day)：** 當日有效 (Rest of Day)
- **IOC (Immediate or Cancel)：** 立即成交否則取消 (Immediate or Cancel)
- **FOK (Fill or Kill)：** 全部成交否則取消 (Fill or Kill)
- **限價單 (Limit Order)：** 以指定價格或更優價格成交
- **市價單 (Market Order)：** 以最佳可成交價格成交（流動性差的股票有風險！）

### 6.3 漲跌幅限制 (Price Limits)
- **一般股票：** 前日收盤價 ±10%
- **ETF：** ±10%
- **權證 (Warrants)：** ±10%（追蹤標的漲跌幅）
- **期貨/選擇權：** 視合約而定（台指期約 ±10-13%）
- **新股上市首五日：** 無漲跌幅限制

### 6.4 檔位 (Tick Size / 升降單位)
| 價格區間 | 最小檔位 |
|----------|----------|
| < NT$10 | NT$0.01 |
| NT$10-50 | NT$0.05 |
| NT$50-100 | NT$0.10 |
| NT$100-500 | NT$0.50 |
| NT$500-1000 | NT$1.00 |
| > NT$1000 | NT$5.00 |

> **檔位摩擦 (Tick Friction)：** NT$1000 以上股票，1 檔 = NT$5 = 0.5%。這意味著小波動策略在高價股上會被檔位吃掉利潤。

### 6.5 檔位摩擦詳解 (Tick Friction Explained)

#### 什麼是檔位摩擦 (What is Tick Friction)？

檔位摩擦是量化交易中一個常被忽略的隱性成本 (Hidden Cost)。台股不像美股可以下任意小數點價格，而是**必須在固定檔位 (Discrete Ticks) 上交易**。你的真實報酬永遠是檔位的整數倍，無法精確反映連續的理論價格。

**英文術語：** 在全球市場分析中，這稱為 **Tick Size Constraint** 或 **Price Discretization**。美國市場在 2000 年代完成「十進制化 (Decimalization)」後，多數股票的最小檔位統一為 USD$0.01，摩擦極低。台灣的階梯式檔位 (Tiered Tick Size) 比美國粗糙很多，這直接影響策略設計。

#### 為什麼重要 (Why It Matters)？

```
假設你預測 2330 會從 NT$1,810 漲到 NT$1,812（模型預測 +0.11%）
→ 但 NT$1,812 不是合法價格（NT$1,000 以上每檔 NT$5）
→ 你只能在 NT$1,810 或 NT$1,815 之間選擇
→ 實際上你無法捕捉這 NT$2 的預測移動
→ 你的模型可能正確，但檔位讓你無法執行 (Execution Gap)

對照美國市場：
AAPL @ USD$200，最小檔 USD$0.01 = 0.005%
→ 幾乎任何預測的移動都可以被執行
→ 模型報酬 ≈ 實際報酬
```

#### 檔位摩擦對策略的影響 (Impact on Strategy Design)

| 策略類型 (Strategy Type) | 檔位影響 | 2330 實例 (NT$1,810) |
|-------------------------|---------|---------------------|
| **剝頭皮 (Scalping)** | 極大 — 利潤常只有 1-2 檔 | 1 檔 NT$5 = 0.28%，扣掉成本 0.21% 後僅剩 0.07% |
| **動量 (Momentum)** | 中等 — 目標 1-3%，檔位佔比小 | 目標 NT$18-54 移動 = 4-11 檔，摩擦可忽略 |
| **均值回歸 (Mean Reversion)** | 中等 — 取決於回歸幅度 | 回歸 0.5% ≈ NT$9 ≈ 2 檔，扣成本後利潤薄 |
| **日內波段 (Intraday Swing)** | 小 — 目標 2-5% | 目標 NT$36-90 = 7-18 檔，摩擦影響極小 |

#### 檔位摩擦的數學表達 (Mathematical Expression)

```
檔位摩擦率 (Tick Friction Rate) = 檔位 (Tick Size) / 股價 (Price)

範例：
  2330  @ NT$1,810 → NT$5 / NT$1,810 = 0.28%
  2317  @ NT$193   → NT$0.5 / NT$193  = 0.26%
  2454  @ NT$1,465 → NT$5 / NT$1,465  = 0.34%
  0050  @ NT$235   → NT$0.5 / NT$235  = 0.21%

對照美股：
  AAPL  @ USD$200  → USD$0.01 / USD$200 = 0.005%（台股的 50 倍精細）
  TSLA  @ USD$360  → USD$0.01 / USD$360 = 0.003%
```

#### 對策 (How to Mitigate)

1. **避開剝頭皮策略 (Avoid Scalping)** — 在台股高價股上，檔位摩擦吃掉大部分利潤。改做 15 分鐘以上的波段
2. **選擇適當標的 (Choose Right Stocks)** — 中價股 (NT$100-500) 的檔位百分比最佳
3. **使用期貨代替 (Use Futures Instead)** — 台指期 1 點 / 32,500 點 = 0.003%，接近美股的精細度
4. **拉大目標波段 (Target Larger Moves)** — 瞄準 1% 以上的移動，讓檔位成本 < 10% 的目標利潤
5. **回測必須模擬檔位 (Backtest Must Simulate Ticks)** — 不要用連續價格計算，要 snap 到最近檔位

## 7. 盤中零股交易 (Intraday Odd Lot Trading)

- 2020 年起實施
- 交易時段：09:00-13:30（與整股相同）
- 最小單位：1 股（整股為 1000股/張）
- 適合小資本策略 (Low Capital Strategy)
- 獨立委託簿 (Order Book)，價差可能較大
- 零股也可當沖
